# 📊 Bitcoin Market Sentiment vs Trader Performance Analysis

## Overview

This project explores the relationship between Bitcoin market sentiment and trader performance using historical trading data from Hyperliquid and the Bitcoin Fear & Greed Index.

The objective is to discover patterns between market psychology and trading profitability using exploratory data analysis and visualization techniques.

---

## Project Objectives

- Analyze Bitcoin market sentiment
- Explore trader profitability
- Compare Buy and Sell trades
- Identify profitable cryptocurrencies
- Visualize trader performance
- Generate actionable insights

---

## Dataset

### Bitcoin Market Sentiment

Contains:

- Date
- Classification
- Fear & Greed Value

### Hyperliquid Historical Trader Data

Contains:

- Account
- Coin
- Execution Price
- Size
- Side
- Closed PnL
- Fee
- Timestamp
- Start Position

## Key Findings

- Traders earned higher average profits during Greed market conditions.
- Buy trades slightly outperformed Sell trades.
- ZRO, SOL, and AVAX ranked among the highest-performing assets.
- Profit distribution contains significant outliers.
- Position size has a strong positive relationship with transaction fees.
